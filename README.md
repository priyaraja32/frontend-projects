#  Smart Task & Productivity Dashboard

A modern task management dashboard built using **React**, **TypeScript**, and **Redux Toolkit**.  
This application allows users to **create, update, delete, and track tasks** with a clean, responsive UI.

---

##  Features

-  Add new tasks
-  Edit existing tasks
-  Delete tasks
-  Track task status (Completed / Pending)
-  Centralized state management with Redux Toolkit
-  Responsive UI using Material UI
-  Accessible components (ARIA-friendly)
-  Unit testing with React Testing Library

---

##  Tech Stack

- **React 18**
- **TypeScript**
- **Redux Toolkit**
- **Material UI**
- **Jest + React Testing Library**
- **Vite**
